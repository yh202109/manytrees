Three Penguin Species

Figure source: allisonhorst.github.io/palmerpenguins

Chinstrap (Pygoscelis antarcticus)

Named for the narrow black band under the chin; slender, long bill; bold facial markings; found mainly on rocky Antarctic islands.

Gentoo (Pygoscelis papua)

Largest of the three species; bright orange-red bill; white patch above eye; fastest swimming penguin (~36 km/h).

Adélie (Pygoscelis adeliae)

Most widespread Antarctic species; short bill with feathering at the base; white eye ring.

Controlling Tree Complexity

Three parameters govern the size of the tree:

Parameter Default Effect
max_depth 5 Maximum splits from root to leaf
min_samples_split 2 Min observations needed to split a node
min_samples_leaf 1 Min observations required in each child

Reduce depth or raise the minimum counts to get a simpler, more interpretable tree.

  body_mass_g ~ bill_length_mm + flipper_length_mm,

Random Forest

From one tree to many

  • Grow \(B\) trees on bootstrap samples
  • Each split considers only \(m\) random features
  • Majority vote (classification) / average (regression)
  • OOB estimate for the generalization error for validation
rf_fit <- random_forest(
  species ~ bill_length_mm + bill_depth_mm +
            flipper_length_mm + body_mass_g,
  data    = penguins_clean,
  n_trees = 200,
  seed    = 8341
)
